# snowflake-geo-performance-optimization
This repo contains my implementation of Snowflake’s Geo Performance guide, focused on spatial query performance and reproducible benchmarking.

## What’s inside
- SQL scripts for setup, loading, and spatial queries (GEOGRAPHY / ST_DISTANCE)
- Performance testing queries + notes
- (Optional) query profile screenshots and results

## How to run
1. Run setup SQL
2. Load data
3. Execute geo queries
4. Compare performance in results
